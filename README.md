# OpenProof — RPO Examples

This repository contains example Registered Probative Objects (RPO)
generated by the OpenProof deterministic simulator.

Each RPO illustrates how institutional reaction bundles can be structured
as sealed, verifiable evidence bundles.

Examples included:

• HR decision governance  
• Institutional crisis response  
• Audit traceability record

Try the simulator:

https://rpo.openproof.net/simulator.html
